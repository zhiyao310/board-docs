## 概述

Lichee Pi 4A 是一款高性能 RISC-V 开发板，搭载平头哥曳影 TH1520 芯片，拥有 4 个玄铁 C910 核心，主频最高 1.85GHz。支持 16GB LPDDR4 内存，板载 Wi-Fi 6、蓝牙 5.2、双千兆网口，适用于 AI 边缘计算、嵌入式开发等场景。

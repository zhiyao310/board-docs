# VisionFive2 Lite

VisionFive 2 Lite is an entry-level RISC-V single-board computer with a low barrier to entry and rich functionality, suitable for education, AIoT, smart home, and industrial IoT scenarios. Powered by the StarFive JH-7110S quad-core processor, the board delivers strong image and video processing capabilities, and features extensive expansion interfaces including PCIe, HDMI, USB 3.0, and Gigabit Ethernet. It is fully compatible with mainstream Linux distributions and open-source toolchains, helping developers achieve rapid development and seamless integration. Whether for learning and research, prototype development, or embedded deployment, VisionFive 2 Lite strikes an excellent balance between performance and cost.
